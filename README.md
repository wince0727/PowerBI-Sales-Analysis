# Power BI Sales Dashboard

## Project Overview
This project analyzes sales data using Power BI to identify trends, performance, and insights.

## Tools Used
- Power BI
- Excel
- SQL (if used)

## Dashboard Features
- Total Sales & Profit
- Category-wise Analysis
- Region-wise Performance
- Monthly Trends

## Files Included
- Power Bi Project.pbix – Power BI report
- AdventureWorks Raw Data – Source data
- dashboard.png – Dashboard preview

## How to Use
1. Download the `.pbix` file.
2. Open with Power BI Desktop.
3. Refresh data if needed.
